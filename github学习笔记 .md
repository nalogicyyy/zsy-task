## 一、GitHub 是什么？
* 自己写的代码可以存在上面，不怕丢
* 别人写的好项目，自己可以免费下载用
* 当改了代码，它会帮我们记录每一次改动
* 多人一起写代码也不会乱
## 二、Git 是什么？
帮我们把本地电脑的代码，传到 GitHub 云端，或者从云端下载到电脑
## 三、github的用处是什么？
### 1.下载别人的项目（最常用）
**流程如下**：  
打开 GitHub 搜到项目  
点绿色按钮 Code  
选 Download ZIP 直接下载压缩包  
解压 → 看 README 说明 → 运行  
这就是最适合小白的方式，不用记任何命令。  
### 2.上传自己写的代码项目保存
**流程如下**：  
在 GitHub 建一个仓库  
用 Git 把本地代码上传上去
## 四、一些简单的命令翻译  
git clone 链接  
翻译：把 GitHub 上的项目，下载到我电脑  
git add .  
翻译：我选中了所有要上传的文件  
git commit -m "第一次提交"  
翻译：给这次改动打个标记，方便以后查找  
git push  
翻译：把本地代码，传到 GitHub 云端  
git pull  
翻译：把云端最新代码，下载到本地
## 五、简单的使用流程
**当想要用别人项目时**：  
搜索、下载 ZIP、解压、看 READM运行即可
**自己想存自己代码时**：  
GitHub 建仓库、打开 Git 、按顺序输 4 行命令上传即可
